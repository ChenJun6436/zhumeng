关于猪码头

About zhumatou

企业介绍：

花生酥缔造者黄老五先生于2015年8月投资创立了住梦网络科技股份有限公司；由西南财经大学经济信息工程学院院长霍伟东先生担任高级顾问，电子科技大学信息与软件工程学院院长秦志光担任高级技术顾问。

畜禽溯源和监管平台猪码头作为住梦公司重点打造项目，坚持以市场为导向，以服务人民群众为核心。项目以生猪监管作为切入点，逐步实现畜禽全监管。


关于猪码头：

猪码头提供生猪产业链追溯平台的信息化技术解决方案，是结合物联网、云技术、大数据和LBS地理信息等技术，通过通讯网络和平台应用，利用RFID和二维码喷墨等设备技术，实现生猪养殖、检疫、屠宰、加工、物流、销售等全过程的无缝跟踪和监管，打通消费者、企业和政府之间的信息壁垒，实现生猪产业的转型升级和其他更多后续市场的拓展。

借助互联网，逐步打通生猪产业链上的各个环节，同时帮助猪产业与其他外部机构实现连接和跨界融合。打破了产业链条割裂、单一产业单打独斗的局面，无缝连接行业对接外部，变外部产业链为内部生态链，形成一个“互联网+猪业”的大平台和生态圈。


我们的愿景：

我们将通过一只专业化的团队去推动互联网+猪业的发展，为消费者、政府部门、市场提供一个透明、安全的生态环境。
